# Garage Invoice Gen

A minimal [Next.js 16](https://nextjs.org/) starter set up with the App Router, TypeScript, Tailwind CSS v4, and ESLint.

## Scripts

```bash
npm run dev
npm run lint
npm run build
```

Open [http://localhost:3000](http://localhost:3000) during development.

## Project Structure

- `src/app/layout.tsx` sets the root layout and metadata.
- `src/app/page.tsx` contains the starter homepage.
- `src/app/globals.css` holds the Tailwind import and global styles.

## Notes

- Node.js `20.9+` is required for Next.js 16.
- The default import alias is `@/*`, mapped to `src/*`.
